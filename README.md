## Project Description
Java Spring Boot web application for recommending recipes to users using XML, XSD, XPath, and XSLT.

## Features
- Read recipes from XML
- Add recipes and users
- Recommend recipes by skill
- Recommend recipes by skill and cuisine
- Display recipes with XSLT
- Filter recipes by cuisine
- View recipe details

## Technologies Used
- Java
- Spring Boot
- Thymeleaf
- XML
- XSD
- XPath
- XSLT
- Jsoup
- Maven

## Data Source
Recipe titles were scraped from the BBC Good Food "Budget Autumn Recipes" page. Cuisine types and difficulty levels were assigned programmatically according to the assignment.
